
## 1. 푸리에 변환 (Fourier Transform)

시간에 대한 함수(Time Domain)를 주파수 성분(Frequency Domain)으로 분해하는 수학적 연산입니다. 임의의 입력 신호를 다양한 주파수를 가진 복소 지수 함수(직교 기저 함수)들의 선형 결합으로 표현합니다. 

$$X(\omega) = \int_{-\infty}^{\infty} x(t) e^{-i\omega t} dt$$

### 1.1. 선형대수학 관점: 직교성과 기저 함수
푸리에 변환의 본질은 미적분학이 아닌 선형대수학(Linear Algebra)에 있습니다.

**1. 벡터 공간과 내적 (Vector Space & Inner Product)**
3차원 공간의 임의의 벡터 $\mathbf{v}$는 세 개의 직교하는 기본 벡터(기저 벡터)인 $\mathbf{\hat{i}}, \mathbf{\hat{j}}, \mathbf{\hat{k}}$의 선형 결합으로 표현할 수 있습니다. 
$$\mathbf{v} = x\mathbf{\hat{i}} + y\mathbf{\hat{j}} + z\mathbf{\hat{k}}$$
여기서 특정 축의 좌표값($x$)을 구하려면 대상 벡터와 해당 기저 벡터를 내적(Dot Product)합니다. 기저 벡터끼리는 서로 직교(Orthogonal)하므로 내적값이 $0$이 되어 사라지고, 자기 자신과의 내적만 남아 해당 축의 성분 크기만 정확히 추출됩니다. ($x = \mathbf{v} \cdot \mathbf{\hat{i}}$)

**2. 함수 공간으로의 확장: 함수의 내적**
함수 $f(t)$를 무한 차원 벡터로 간주하면, 연속적인 함수 공간에서의 내적은 두 함수를 곱해서 적분하는 연산으로 정의됩니다.
$$\langle f, g \rangle = \int_{-\infty}^{\infty} f(t)g^*(t) dt$$
이 적분 결과가 $0$이라면 "두 함수 $f(t)$와 $g(t)$는 서로 직교한다"고 수학적으로 정의합니다.

**3. 복소 지수 함수의 직교성**
푸리에 변환은 오일러 공식에 의해 삼각함수와 동치인 복소 지수 함수 $e^{i\omega t}$를 기저 함수로 사용합니다. 서로 다른 주파수 $\omega_1$과 $\omega_2$를 가진 두 복소 지수 함수의 내적은 다음과 같습니다.
$$\int_{-\infty}^{\infty} e^{i\omega_1 t} e^{-i\omega_2 t} dt = \int_{-\infty}^{\infty} e^{i(\omega_1 - \omega_2)t} dt$$
* **주파수가 다를 때 ($\omega_1 \neq \omega_2$):** 진동하면서 양의 넓이와 음의 넓이가 정확히 상쇄되어 적분값이 $0$이 됩니다. (직교함)
* **주파수가 같을 때 ($\omega_1 = \omega_2$):** $e^0 = 1$이 되므로 적분값은 $0$이 아닌 무한대가 됩니다.

**4. 푸리에 변환의 수학적 본질: 투영(Projection)**
푸리에 변환 식은 임의의 신호 $x(t)$와 특정 주파수 $\omega$를 가진 기저 함수 $e^{i\omega t}$를 내적하는 식입니다. 복잡한 신호를 무한히 많은 주파수 축에 투영(Projection)시켜, 각 주파수 축에 해당하는 성분(계수)을 추출해 내는 엄밀한 분해 과정입니다.

## 2. 라플라스 변환 (Laplace Transform)

라플라스 변환은 지수적 감쇠 또는 발산 성분($\sigma$)을 추가하여 복소 주파수 도메인($s$-domain)으로 신호를 매핑합니다. (복소 변수 $s = \sigma + i\omega$)

$$F(s) = \int_{0}^{\infty} f(t) e^{-st} dt$$

### 2.1. 미분 방정식을 대수학으로 변환
미분 연산이 단순히 복소 변수 $s$를 곱하는 사칙연산(대수 방정식)으로 바뀝니다. 라플라스 변환 연산자 $\mathcal{L}$은 '방정식 전체(좌항과 우항)'에 동일하게 적용됩니다.

$$\mathcal{L}\{f'(t)\} = sF(s) - f(0)$$

**1차 미분 방정식 풀이 상세 과정:**
방정식 $\frac{dv(t)}{dt} + 3v(t) = 2$ ($v(0) = 0$) 가 있다고 가정합니다. 양변에 변환을 적용합니다.
$$\mathcal{L} \left\{ \frac{dv(t)}{dt} + 3v(t) \right\} = \mathcal{L} \{ 2 \}$$

* **좌항 변환:** 미분 정리와 선형성에 의해 $sV(s) - v(0) + 3V(s)$가 됩니다.
* **우항 변환 ($\frac{2}{s}$ 도출):** 상수 $2$는 시간에 대해 변하지 않는 함수 $f(t) = 2$입니다. 
  $$\mathcal{L}\{2\} = \int_{0}^{\infty} 2 e^{-st} dt = \left[ -\frac{2}{s} e^{-st} \right]_{0}^{\infty} = \frac{2}{s}$$

식을 정리하면 다음과 같습니다.
1. $sV(s) - v(0) + 3V(s) = \frac{2}{s}$
2. 초기값 $0$ 대입 후 $V(s)$로 묶기: $V(s)(s + 3) = \frac{2}{s}$
3. 최종 $V(s)$ 도출: $V(s) = \frac{2}{s(s+3)}$

### 2.2. 전달 함수 (Transfer Function) 도출
전달 함수는 입력에 대한 출력의 비율($H(s) = \frac{Y(s)}{U(s)}$)이며, 대상 시스템의 물리적 법칙으로부터 직접 유도합니다.

**드론 모터-프로펠러 시스템 유도 예시:**
뉴턴 제2법칙(관성 $J$, 마찰 $B$, 입력 토크 $T(t)$, 출력 각속도 $\omega(t)$) 적용:
$$J \frac{d\omega(t)}{dt} + B\omega(t) = T(t)$$
초기 상태가 정지($\omega(0) = 0$)되어 있다고 가정하고 양변을 라플라스 변환 및 정리:
$$J(s\Omega(s)) + B\Omega(s) = \mathbf{T}(s)$$
$$H(s) = \frac{\Omega(s)}{\mathbf{T}(s)} = \frac{1}{Js + B}$$

### 2.3. 역 라플라스 변환 (Inverse Laplace Transform)과 해의 의미
$s$-도메인에서 구한 대수적 해(예: $Y(s) = \frac{10}{s(s+5)}$)를 다시 시간 $t$ 도메인의 실제 물리적 움직임인 $y(t)$로 복원하는 과정입니다. 

소프트웨어적으로나 수학적으로 미분 방정식을 직접 적분해서 푸는 것보다 효율적이기 때문에, $s$-도메인의 분수 형태로 식을 정리한 후 **부분분수 전개(Partial Fraction Expansion)**를 통해 수식을 쪼개고 미리 증명된 '역변환 테이블'을 매칭하여 해를 구합니다. $Y(s)$ 자체는 물리적 해석이 어려운 대수적 상태일 뿐, 미분을 회피하기 위한 '수학적 중간 정거장' 데이터 역할을 합니다.

### 2.4. 극점 (Pole), 영점 (Zero), 그리고 역변환의 함정 ($e^{-st}$ vs $e^{pt}$)
* **극점 (Pole):** 전달 함수의 분모를 $0$으로 만드는 $s$의 값입니다. 
* **영점 (Zero):** 전달 함수의 분자를 $0$으로 만드는 $s$의 값입니다.

**수학적 혼동 주의: 왜 $e^{-st}$ 식을 쓰는데 극점이 양수면 발산하는가?**
1. **변환 도구로서의 $e^{-st}$:** 적분식 $\int f(t) e^{-st} dt$의 $e^{-st}$는 단순히 함수 $f(t)$를 $s$-도메인으로 가져가기 위해 곱하는 필터입니다.
2. **해답으로서의 $e^{pt}$:** 역변환을 통해 시스템의 최종적인 시간 도메인 해답 $y(t)$를 구하면, 해답은 각 극점($p$)을 지수로 가지는 항들의 합(예: $y(t) = A e^{p_1 t} + B e^{p_2 t}$)으로 나타납니다.
따라서, 극점 $p$의 실수부가 양수가 되면 $e^{pt}$의 지수부가 양수가 되어 시스템의 출력이 시간에 따라 기하급수적으로 발산(폭주)합니다. 극점이 음수(좌반면)여야 $0$으로 안전하게 수렴합니다.

### 2.5. PID 제어: 극점 이동의 수학적 증명
불안정한 시스템 $H(s) = \frac{1}{s^2 - 1}$ (극점 $s=1, -1$) 이 있다고 가정합니다.
폐루프 제어 시스템의 안정성을 결정하는 분모를 **특성 방정식**이라 부르며 $1 + C(s)H(s) = 0$ 입니다.
PD 제어기 $C(s) = K_p + K_d s$ 를 대입하고 정리하면 다음과 같은 새로운 2차 방정식이 됩니다.
$$s^2 + K_d s + (K_p - 1) = 0$$
근의 공식으로 구한 새로운 극점 $s = \frac{-K_d \pm \sqrt{K_d^2 - 4(K_p - 1)}}{2}$ 에서, 코드로 $K_d > 0$ 이고 $K_p > 1$ 이 되도록 게인을 튜닝하면, 극점의 실수부가 무조건 음수 영역(좌반면)으로 강제 이동하게 되며 시스템이 안정화됩니다.

## 3. 이산 시스템과 고속 푸리에 변환 (FFT)

### 3.1. 이산 푸리에 변환(DFT)과 시간 복잡도
디지털 데이터는 적분 대신 유한 구간의 덧셈($\sum$)을 수행하는 이산 푸리에 변환(DFT)을 사용합니다.
$$X[k] = \sum_{n=0}^{N-1} x[n] e^{-i \frac{2\pi}{N} k n}$$
단 하나의 주파수 $X[k]$를 구하기 위해 $N$번 반복하며, 이를 $N$개의 주파수에 대해 수행하므로 총 연산량은 명확하게 이중 루프 구조인 $O(N^2)$이 됩니다.

### 3.2 이산 푸리에 변환(DFT)의 정의와 한계

배열의 인덱스를 $n$, 총 데이터의 개수를 $N$이라고 할 때, 이산 시간에서의 푸리에 변환 식은 다음과 같이 변형됩니다.

- 연속 시간 변환: $X(\omega) = \int_{-\infty}^{\infty} x(t) e^{-i\omega t} dt$
    
- 이산 시간 변환: $X[k] = \sum_{n=0}^{N-1} x[n] e^{-i \frac{2\pi}{N} k n}$
    
- $x[n]$: 입력 데이터 배열의 $n$번째 값 (시간 도메인)
    
- $X[k]$: 변환된 결과 배열의 $k$번째 값 (주파수 도메인)
    
- $e^{-i \frac{2\pi}{N} k n}$: 이산 공간에서의 복소 지수 기저 함수입니다. 연속 공간의 주파수 $\omega$가 이산 공간에서는 $\frac{2\pi}{N} k$ 로 대체되었습니다. $N$개의 샘플을 한 바퀴($2\pi$)로 나누어 계산한다는 의미입니다.

### 3.3. 고속 푸리에 변환(FFT)의 알고리즘 유도
FFT(Cooley-Tukey 알고리즘)는 배열을 짝수 그룹($E[k]$)과 홀수 그룹($O[k]$)으로 나누는 분할 정복을 사용하여 연산량을 $O(N \log N)$으로 줄입니다.
$$X[k] = E[k] + W_N^k O[k]$$

**절반 인덱스($N/2$) 부호 반전의 완벽한 증명:**
주파수 인덱스가 절반($N/2$)을 넘어갈 때의 회전 인자 $W_N^{k + N/2} = W_N^k \cdot W_N^{N/2}$ 를 분석합니다.
1. **수학적 전개:** $W_N^{N/2} = (e^{-i\frac{2\pi}{N}})^{N/2} = e^{-i\pi}$. 오일러 공식에 의해 $\cos(-\pi) + i\sin(-\pi) = -1$ 이 됩니다.
2. **기하학적 증명:** 복소평면 단위 원에서 지수에 $N/2$가 곱해졌다는 것은 원의 절반(180도, $\pi$) 회전을 의미합니다. 원점 대칭이 되므로 원래 벡터에 $-1$을 곱한 것과 같습니다.
결과적으로 기존 식은 다음과 같이 뺄셈으로 치환됩니다.
$$X[k + \frac{N}{2}] = E[k] - W_N^k O[k]$$
새로 계산하지 않고 앞선 값을 재사용하여 부호만 뒤집어 덮어씁니다(나비 연산).

### 3.4. 역변환 (IFFT) 메커니즘
역변환(IDFT)은 주파수 도메인을 시간 도메인으로 복원합니다.
$$x[n] = \frac{1}{N} \sum_{k=0}^{N-1} X[k] e^{i \frac{2\pi}{N} k n}$$
지수부 부호가 양수(+)이고 $1/N$ 스케일링이 추가됩니다. 알고리즘 코드는 정방향 FFT를 재사용합니다.
1. 입력 주파수 배열에 켤레 복소수를 취합니다. (허수부 부호 반전)
2. 정방향 FFT 실행 ($O(N \log N)$).
3. 결과에 다시 켤레 복소수를 취하고 $N$으로 나눕니다.

### 3.5. FFT 코드 레벨 구현과 한계 분석
```python
import numpy as np

def recursive_fft(x):
    N = len(x)
    if N <= 1: return x
    
    even = recursive_fft(x[0::2]) # 비효율: Array Slicing 메모리 오버헤드
    odd  = recursive_fft(x[1::2])
    
    X = np.zeros(N, dtype=np.complex128)
    for k in range(N // 2): # 절반까지만 순회
        t = np.exp(-2j * np.pi * k / N) * odd[k]
        
        X[k] = even[k] + t
        X[k + N // 2] = even[k] - t # 부호 반전 재사용 
        
    return X
````

**현업의 구조적 개선:** 재귀 배열 복사에 따른 콜스택 부하를 막기 위해, 배열 인덱스를 비트 단위로 뒤집어 재배치(Bit-reversal)한 뒤 반복문을 사용하여 메모리 주소 내에서 직접 맞바꾸는 **제자리 연산(In-place Algorithm)**을 수행합니다.

## 4. 라그랑주 방정식 (Lagrange's Equations)

뉴턴 역학을 힘과 벡터가 아닌, 에너지(스칼라량)를 기반으로 재구성한 방정식입니다. 라그랑지안 $L = T(운동) - V(위치)$일 때, 일반화 좌표 $q_i$에 대한 운동 방정식은 다음과 같습니다.

$$\frac{d}{dt} \left( \frac{\partial L}{\partial \dot{q}_i} \right) - \frac{\partial L}{\partial q_i} = 0$$

도출된 비선형 상미분 방정식을 코드로 옮긴 후 수치 해석 알고리즘(RK4 등)으로 시뮬레이션합니다. (물리 엔진, 로봇 역운동학)

## 5. 개념 검증: 실전 예제 및 풀이

### 5.1. 이산 푸리에 변환 (DFT) - DC 성분 추출

**문제:** 배열 $x = [3, 3, 3, 3]$ (총 길이 $N=4$)을 푸리에 변환했을 때, 주파수 인덱스가 $0$인 성분 $X[0]$의 값을 구하시오.

**풀이 및 의미:**

$$X[0] = \sum_{n=0}^{3} x[n] e^{-i \frac{2\pi}{N} \cdot 0 \cdot n} = \sum_{n=0}^{3} x[n] \cdot 1$$

$e^0 = 1$이 되므로 $3+3+3+3 = 12$ 가 도출됩니다. 주파수가 $0$Hz인 이 성분을 신호의 **DC 성분(직류 성분 또는 평균값)** 이라 부릅니다. 개발할 때 배열 요소의 총합이나 평균을 구하는 루프는 푸리에 변환의 $0$Hz 성분 추출과 완벽히 동치인 연산입니다.

### 5.2. 라플라스 변환 - 미분 방정식 치환

**문제:** $\frac{dy(t)}{dt} + 5y(t) = 10$, 초기 상태 $y(0) = 0$ 일 때 라플라스 변환을 통해 $Y(s)$를 도출하시오.

**풀이 및 의미:** 양변에 라플라스 연산자 $\mathcal{L}$ 적용:

$$sY(s) - y(0) + 5Y(s) = \frac{10}{s}$$

초기값 $0$ 대입 후 $Y(s)$로 묶기:

$$Y(s)(s + 5) = \frac{10}{s}$$

$$Y(s) = \frac{10}{s(s+5)}$$

복잡한 미분 방정식을 사칙연산 이항만으로 완벽한 대수 다항식 형태로 도출했습니다.

### 5.3. 전달 함수와 극점 - 시스템 안정성 판별

**문제:** 모터 제어 시스템의 전달 함수 $H(s) = \frac{s + 2}{s^2 - 2s - 8}$ 의 극점을 구하고 시스템의 안정성을 판별하시오.

**풀이 및 의미:** 분모를 $0$으로 만드는 방정식: $s^2 - 2s - 8 = 0$

인수분해: $(s-4)(s+2) = 0$

**극점(Pole):** $s = 4, -2$

**안정성 판별:** 역 라플라스 변환 시 시간 도메인 시스템 해답은 $e^{4t}$ 와 $e^{-2t}$ 항으로 전개됩니다. 여기서 극점이 양수인 $4$를 포함하고 있으므로, 시간 $t$가 지남에 따라 $e^{4t}$가 기하급수적으로 폭주하게 되어 이 시스템은 **불안정**합니다.

## 6. 핵심 공학/수학 용어 사전

- **동치 (Equivalence):** 논리적으로 완전히 같음을 의미. 형태는 달라도 결과적으로 동일한 연산이나 정보를 담고 있을 때 사용 (예: 오일러 공식 기반 삼각함수와 복소 지수 함수 변환).
    
- **켤레 복소수 (Complex Conjugate):** 허수부의 부호만 반대로 바꾼 수. 켤레 복소수끼리 곱하면 허수부가 사라짐. 역변환(IFFT) 시 회전 방향을 뒤집는 트릭으로 사용.
    
- **회전 인자 (Twiddle Factor, $W_N$):** 복소평면 단위 원을 $N$등분하여 특정 인덱스만큼 회전하는 위치 상수($W_N = e^{-i\frac{2\pi}{N}}$). FFT 나비 연산의 뼈대.
    
- **직교성 (Orthogonality):** 두 벡터나 함수를 내적(적분)했을 때 결과가 $0$이 되는 성질. 신호가 서로 간섭하지 않고 독립적으로 분리될 수 있는 수학적 근거.
    
- **기저 함수 (Basis Function):** 공간의 모든 점을 표현하기 위해 사용하는 직교하는 기본 단위 함수들.
    
- **내적 (Inner Product / Dot Product):** 요소들을 곱해 더하거나 적분하여 스칼라 값으로 뽑아내는 연산. 특정 기저 함수 패턴이 얼마나 포함되어 있는지 추출하는 투영(Projection) 과정.
    
- **도메인 (Domain):** 변수가 속하여 변화를 나타내는 기준 축(시간, 주파수, 복소 주파수 $s$-도메인).
    
- **이산 (Discrete) vs 연속 (Continuous):** 미적분 대상인 무한한 실수 흐름(연속)과, 루프 알고리즘 대상인 정수로 셀 수 있는 배열 상태(이산).
    
- **제자리 연산 (In-place Algorithm):** 추가 메모리 할당 없이, 배열 내부 주소 내에서 원소들끼리 위치를 맞바꾸어 결과를 도출하는 메모리 최적화 기법.
    
- **전달 함수 (Transfer Function):** 초기 상태 $0$에서 입력 신호 라플라스 변환에 대한 출력 신호 라플라스 변환 비율. 시스템 반응을 결정하는 수학적 뼈대.
    
- **극점 (Pole)과 영점 (Zero):** 분모를 $0$으로 만드는 값(안정성 결정)과 분자를 $0$으로 만드는 값(반응 파형 영향).
    
- **특성 방정식 (Characteristic Equation):** 폐루프 시스템 전체 전달 함수의 분모를 $0$으로 두는 방정식. PID 튜닝으로 해를 이동시키는 타겟.
    
- **폐루프 (Closed-loop) / 피드백 (Feedback):** 현재 시스템 상태를 센서로 읽어와 목표값과의 오차를 구하고, 이를 다음 명령에 반영하는 순환 구조.
