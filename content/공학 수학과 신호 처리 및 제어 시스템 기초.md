## 1. 푸리에 변환 (Fourier Transform)


시간에 대한 함수(Time Domain)를 주파수 성분(Frequency Domain)으로 분해하는 수학적 연산입니다. 임의의 입력 신호를 다양한 주파수를 가진 복소 지수 함수(직교 기저 함수)들의 선형 결합으로 표현합니다. 

$$X(\omega) = \int_{-\infty}^{\infty} x(t) e^{-i\omega t} dt$$

### 1.1. 선형대수학 관점: 직교성과 기저 함수
푸리에 변환의 본질은 미적분학이 아닌 선형대수학(Linear Algebra)에 있습니다.

**1. 벡터 공간과 내적 (Vector Space & Inner Product)**

3차원 공간의 임의의 벡터 $\mathbf{v}$는 세 개의 직교하는 기본 벡터(기저 벡터)인 $\mathbf{\hat{i}}, \mathbf{\hat{j}}, \mathbf{\hat{k}}$의 선형 결합으로 표현할 수 있습니다. 
$$\mathbf{v} = x\mathbf{\hat{i}} + y\mathbf{\hat{j}} + z\mathbf{\hat{k}}$$
여기서 특정 축의 좌표값($x$)을 구하려면 대상 벡터와 해당 기저 벡터를 내적(Dot Product)합니다. 기저 벡터끼리는 서로 직교(Orthogonal)하므로 내적값이 $0$이 되어 사라지고, 자기 자신과의 내적만 남아 해당 축의 성분 크기만 정확히 추출됩니다.
$$x = \mathbf{v} \cdot \mathbf{\hat{i}}$$

**2. 함수 공간으로의 확장: 함수의 내적**
함수 $f(t)$를 무한 차원 벡터로 간주하면, 연속적인 함수 공간에서의 내적은 두 함수를 곱해서 적분하는 연산으로 정의됩니다.
$$\langle f, g \rangle = \int_{-\infty}^{\infty} f(t)g^*(t) dt$$
이 적분 결과가 $0$이라면 "두 함수 $f(t)$와 $g(t)$는 서로 직교한다"고 수학적으로 정의합니다.

**3. 복소 지수 함수의 직교성**

푸리에 변환은 오일러 공식에 의해 삼각함수와 동치인 복소 지수 함수 $e^{i\omega t}$를 기저 함수로 사용합니다. 서로 다른 주파수 $\omega_1$과 $\omega_2$를 가진 두 복소 지수 함수의 내적을 계산해 보면 다음과 같습니다.
$$\int_{-\infty}^{\infty} e^{i\omega_1 t} e^{-i\omega_2 t} dt = \int_{-\infty}^{\infty} e^{i(\omega_1 - \omega_2)t} dt$$
* **주파수가 다를 때 ($\omega_1 \neq \omega_2$):** 코사인과 사인 파동이 진동하면서 양의 넓이와 음의 넓이가 정확히 상쇄되어 적분값이 $0$이 됩니다. (직교함)
* **주파수가 같을 때 ($\omega_1 = \omega_2$):** $e^0 = 1$이 되므로 적분값은 $0$이 아닌 무한대가 됩니다.
즉, 서로 다른 주파수를 가진 모든 $e^{i\omega t}$ 함수들은 무한 차원 공간에서 서로 완벽하게 직교하는 기저 벡터들입니다.

**4. 푸리에 변환의 수학적 본질: 투영(Projection)**
푸리에 변환 식은 임의의 신호 $x(t)$와 특정 주파수 $\omega$를 가진 기저 함수 $e^{i\omega t}$를 내적하는 식입니다. 시간 도메인의 복잡한 신호 $x(t)$를 무한히 많은 주파수 축에 투영(Projection)시켜, 각 주파수 축에 해당하는 성분(계수)을 추출해 내는 엄밀한 선형대수학적 분해 과정입니다.

## 2. 라플라스 변환 (Laplace Transform)


푸리에 변환이 진동하는 주파수 성분($\omega$)만 다룬다면, 라플라스 변환은 지수적 감쇠 또는 발산 성분($\sigma$)을 추가하여 복소 주파수 도메인($s$-domain)으로 신호를 매핑합니다. (복소 변수 $s = \sigma + i\omega$)

$$F(s) = \int_{0}^{\infty} f(t) e^{-st} dt$$

### 2.1. 미분 방정식을 대수학으로 변환
시간 도메인에서는 미분과 적분으로 계산해야 하지만, 변환을 거치면 미분 연산이 단순히 복소 변수 $s$를 곱하는 사칙연산(대수 방정식)으로 바뀝니다. 라플라스 변환 연산자 $\mathcal{L}$은 '방정식 전체(좌항과 우항)'에 동일하게 적용됩니다.

$$\mathcal{L}\{f'(t)\} = sF(s) - f(0)$$

**1차 미분 방정식 풀이 상세 과정:**
모터의 속도 $v(t)$에 대한 방정식 $\frac{dv(t)}{dt} + 3v(t) = 2$ ($v(0) = 0$) 가 있다고 가정합니다. 양변에 변환을 적용합니다.
$$\mathcal{L} \left\{ \frac{dv(t)}{dt} + 3v(t) \right\} = \mathcal{L} \{ 2 \}$$

* **좌항 변환:** 미분 정리와 선형성에 의해 $sV(s) - v(0) + 3V(s)$가 됩니다.
* **우항 변환 ($\frac{2}{s}$ 도출):** 상수 $2$는 시간에 대해 변하지 않는 함수 $f(t) = 2$입니다. 변환 정의에 대입하면 다음과 같습니다.
  $$\mathcal{L}\{2\} = \int_{0}^{\infty} 2 e^{-st} dt = \left[ -\frac{2}{s} e^{-st} \right]_{0}^{\infty}$$
  $t \to \infty$일 때 $e^{-\infty} = 0$, $t=0$일 때 $e^0 = 1$이 되므로, $0 - \left(-\frac{2}{s} \cdot 1\right) = \frac{2}{s}$ 가 도출됩니다.

식을 정리하면 다음과 같습니다.
1. $sV(s) - v(0) + 3V(s) = \frac{2}{s}$
2. 초기값 $0$ 대입 후 $V(s)$로 묶기: $V(s)(s + 3) = \frac{2}{s}$
3. 최종 $V(s)$ 도출: $V(s) = \frac{2}{s(s+3)}$

### 2.2. 전달 함수 (Transfer Function) 도출


전달 함수는 입력에 대한 출력의 비율($H(s) = \frac{Y(s)}{U(s)}$)이며, 임의로 정하는 것이 아니라 대상 시스템의 물리적 법칙으로부터 직접 유도합니다.

**드론 모터-프로펠러 시스템 유도 예시:**
뉴턴의 제2법칙(회전 운동)에 의해, 모터 인가 토크(입력 $T(t)$)는 관성 모멘트($J$)에 의한 저항력과 공기 마찰계수($B$)에 의한 저항력의 합과 같습니다. (출력 각속도 $\omega(t)$)
$$J \frac{d\omega(t)}{dt} + B\omega(t) = T(t)$$
초기 상태가 정지($\omega(0) = 0$)되어 있다고 가정하고 양변을 라플라스 변환합니다.
$$J(s\Omega(s) - 0) + B\Omega(s) = \mathbf{T}(s)$$
$$\Omega(s)(Js + B) = \mathbf{T}(s)$$
입력 대비 출력의 비율로 식을 정리하여 물리 엔진의 뼈대가 되는 전달 함수를 얻어냅니다.
$$H(s) = \frac{\Omega(s)}{\mathbf{T}(s)} = \frac{1}{Js + B}$$

### 2.3. 극점 (Pole)과 영점 (Zero)

* **극점 (Pole):** 전달 함수의 분모를 $0$으로 만드는 $s$의 값입니다. 실수부 $\sigma$가 음수(좌반면)면 시간이 지날수록 진동이 감쇠하고 멈추며(안정), 양수(우반면)면 지수적으로 증폭되어 발산(폭주)합니다.
* **영점 (Zero):** 전달 함수의 분자를 $0$으로 만드는 $s$의 값입니다.

### 2.4. PID 제어: 극점 이동의 수학적 증명


불안정한 시스템 $H(s) = \frac{1}{s^2 - 1}$ 이 있다고 가정합니다. 분모 $s^2 - 1 = 0$ 을 만족하는 극점이 $s = 1, -1$ 이며, $1$(양수) 때문에 시스템은 폭주합니다.

이를 제어하기 위해 센서 피드백 구조(폐루프)와 제어기 $C(s)$를 달아줍니다. 전체 시스템의 안정성을 결정하는 새로운 분모를 **특성 방정식(Characteristic Equation)**이라고 부르며 $1 + C(s)H(s) = 0$ 입니다.

여기에 PD 제어기 $C(s) = K_p + K_d s$ 를 대입합니다.
$$1 + (K_p + K_d s) \frac{1}{s^2 - 1} = 0$$
양변에 $(s^2 - 1)$을 곱하여 2차 방정식으로 정리합니다.
$$s^2 + K_d s + (K_p - 1) = 0$$
근의 공식으로 구한 새로운 극점 $s$는 다음과 같습니다.
$$s = \frac{-K_d \pm \sqrt{K_d^2 - 4(K_p - 1)}}{2}$$
개발자가 코드 상에서 $K_d > 0$ 이고 $K_p > 1$ 이 되도록 게인 값을 튜닝하면, 위 식의 실수부가 무조건 음수 영역(좌반면)으로 강제 이동하게 되며, 시스템은 발산하지 않고 $0$으로 수렴(안정화)합니다.

## 3. 이산 시스템과 고속 푸리에 변환 (FFT)

### 3.1. 이산 푸리에 변환(DFT)과 $O(N^2)$의 시간 복잡도
컴퓨터 메모리 상의 데이터는 유한하고 불연속적이므로 적분($\int$) 대신 유한 구간의 덧셈($\sum$)을 수행하는 이산 푸리에 변환(DFT)을 사용합니다.
$$X[k] = \sum_{n=0}^{N-1} x[n] e^{-i \frac{2\pi}{N} k n}$$

이 수식을 코드로 구현하면 다음과 같은 이중 루프 구조가 됩니다.
```python
for k in range(N):              # 외부 루프: N개의 주파수 성분 도출
    for n in range(N):          # 내부 루프: 입력 배열 x 전체를 순회하며 내적
        basis_function = exp(-1j * (2 * pi / N) * k * n) 
        X[k] += x[n] * basis_function
````

단 하나의 $X[k]$를 구하기 위해 $N$번 반복하며, 이를 $N$개의 주파수에 대해 수행하므로 총 연산량은 명확하게 $N \times N = N^2$이 됩니다.

### 3.2. 고속 푸리에 변환(FFT)의 알고리즘 유도

FFT(Cooley-Tukey 알고리즘)는 배열을 짝수 그룹($E[k]$)과 홀수 그룹($O[k]$)으로 나누는 분할 정복(Divide and Conquer)을 사용하여 연산량을 $O(N \log N)$으로 줄입니다.

$$X[k] = E[k] + W_N^k O[k]$$

(단, $W_N = e^{-i \frac{2\pi}{N}}$ 은 회전 인자(Twiddle Factor))

**절반 인덱스($N/2$) 부호 반전의 완벽한 증명:**

주파수 인덱스가 절반($N/2$)을 넘어가는 시점인 $W_N^{k + N/2}$ 의 값을 구하는 과정입니다. 지수 법칙에 의해 $W_N^k \cdot W_N^{N/2}$ 로 분리됩니다. 여기서 $W_N^{N/2}$만 계산해 봅니다.

1. **수학적 전개 (오일러 공식):**
    
    $$W_N^{N/2} = \left( e^{-i\frac{2\pi}{N}} \right)^{\frac{N}{2}} = e^{-i\pi}$$
    
    오일러 공식($e^{i\theta} = \cos\theta + i\sin\theta$)에 $-\pi$를 대입합니다. $\cos(-\pi)$는 $-1$이고, $\sin(-\pi)$는 $0$이므로 $e^{-i\pi} = -1$ 이 도출됩니다.
    
2. **기하학적 증명 (180도 회전):**
    
    회전 인자 $W_N$은 복소평면의 단위 원 위를 시계 방향으로 도는 위치를 의미합니다. 지수에 $N/2$가 곱해졌다는 것은 총 스텝 $N$ 중 절반을 이동했다는 뜻이므로 정확히 180도($\pi$) 회전합니다. 복소평면에서 180도 회전은 실수부와 허수부의 부호가 모두 바뀌는 원점 대칭이므로, 원래 벡터에 $-1$을 곱한 것과 완벽히 같습니다.
    

결과적으로 $W_N^{k + N/2} = -W_N^k$ 가 성립하며, 기존 변환 식은 다음과 같이 뺄셈으로 치환됩니다.

$$X[k + \frac{N}{2}] = E[k] - W_N^k O[k]$$

이 수학적 성질(나비 연산, Butterfly Operation) 덕분에 뒤쪽 절반의 배열을 구할 때는 무거운 복소수 곱셈을 다시 할 필요 없이, 앞서 구한 값을 재사용하여 부호만 뒤집어 덮어쓰면 됩니다.

### 3.3. 역변환 (IFFT) 메커니즘

역변환(IDFT)은 주파수 도메인을 시간 도메인으로 복원하는 과정입니다.

$$x[n] = \frac{1}{N} \sum_{k=0}^{N-1} X[k] e^{i \frac{2\pi}{N} k n}$$

정방향 변환과 비교하여 두 가지 차이점이 있습니다. 1) 지수부의 부호가 양수(+)이고, 2) $1/N$ 스케일링이 추가됩니다.

이를 코드에서 $O(N \log N)$으로 처리하기 위해 정방향 FFT 함수를 재사용합니다.

1. 입력 주파수 배열 $X$의 각 요소에 켤레 복소수를 취합니다. (허수부 부호 반전으로 지수부 양수 효과 도출)
    
2. 정방향 FFT 함수를 그대로 실행합니다.
    
3. 결과 배열에 다시 켤레 복소수를 취하고, 배열 전체를 $N$으로 나눕니다.
    

### 3.4. FFT 코드 레벨 구현과 한계 분석


``` python
import numpy as np

def recursive_fft(x):
    N = len(x)
    if N <= 1: return x
    
    even = recursive_fft(x[0::2]) # 비효율: Array Slicing에 의한 메모리 오버헤드
    odd  = recursive_fft(x[1::2])
    
    X = np.zeros(N, dtype=np.complex128) # 비효율: 지속적인 메모리 할당
    for k in range(N // 2): # N 전체가 아닌 N/2 까지만 루프 순회
        t = np.exp(-2j * np.pi * k / N) * odd[k]
        
        X[k] = even[k] + t
        X[k + N // 2] = even[k] - t # 부호 반전 수학 원리가 뺄셈으로 치환
        
    return X
```

**현업 시스템 관점의 개선:**

위 하향식(Top-down) 재귀 코드는 배열 복사(Slicing)와 메모리 할당 오버헤드, 콜스택 부하를 유발합니다. 실제 백엔드나 임베디드 라이브러리(FFTW 등)는 배열의 인덱스를 비트 단위로 뒤집어(Bit-reversal) 재배치해 둔 뒤, 재귀 없이 상향식(Bottom-up) 반복문을 사용하여 원본 배열 내에서 요소들끼리 위치만 맞바꾸는 **제자리 연산(In-place Algorithm)** 구조로 개선하여 사용합니다.

## 4. 라그랑주 방정식 (Lagrange's Equations)

뉴턴 역학($\mathbf{F} = m\mathbf{a}$)을 힘과 벡터가 아닌, 에너지(스칼라량)를 기반으로 재구성한 해석 역학의 핵심 방정식입니다.

시스템의 라그랑지안 $L$을 운동 에너지 $T$와 위치 에너지 $V$의 차이($L = T - V$)로 정의할 때, 임의의 일반화 좌표 $q_i$에 대한 운동 방정식은 다음과 같습니다.

$$\frac{d}{dt} \left( \frac{\partial L}{\partial \dot{q}_i} \right) - \frac{\partial L}{\partial q_i} = 0$$

구속 조건이 복잡한 다물체 시스템에서 벡터 기반 힘 계산의 어려움을 일반화 좌표계를 도입하여 극복합니다. 도출된 비선형 상미분 방정식은 해석적으로 풀기 어려우므로, 코드로 옮긴 후 Runge-Kutta(RK4)와 같은 수치 해석 알고리즘(ODE Solver)으로 시스템 상태를 시뮬레이션합니다. (로봇 팔 역운동학, 물리 엔진 구현 등에 사용)

## 5. 핵심 공학/수학 용어 사전

- **동치 (Equivalence):** 두 수학적 명제나 수식이 논리적으로 완전히 같음을 의미합니다. 수식의 형태는 달라도 결과적으로 완전히 동일한 정보나 연산을 담고 있을 때 사용합니다. (예: 오일러 공식을 통한 삼각함수와 복소 지수 함수의 변환)
    
- **켤레 복소수 (Complex Conjugate):** 복소수 $z = a + bi$ 에서 허수부의 부호만 반대로 바꾼 수 $z^* = a - bi$ 입니다. 복소수와 그 켤레 복소수를 곱하면 허수부가 사라지고 순수한 크기(실수)의 제곱만 남습니다. 역변환(IFFT) 시 회전 방향을 뒤집는 트릭으로 사용됩니다.
    
- **회전 인자 (Twiddle Factor, $W_N$):** 복소평면 상의 단위 원을 $N$등분 했을 때, 특정 인덱스만큼 원 위를 회전하는 위치를 나타내는 복소 지수 상수($W_N = e^{-i\frac{2\pi}{N}}$)입니다. FFT 나비 연산의 핵심 뼈대입니다.
    
- **직교성 (Orthogonality):** 두 벡터나 연속 함수를 내적(또는 적분)했을 때 그 결과가 정확히 $0$이 되는 성질입니다. 두 신호가 서로 간섭하지 않고 독립적으로 분리될 수 있는 수학적 근거가 됩니다.
    
- **기저 함수 (Basis Function):** 특정 수학적 공간의 모든 점을 표현하기 위해 사용하는, 서로 직교하는 가장 기본적인 단위 함수들입니다. (공간 좌표계의 $x, y, z$ 축과 동일한 역할)
    
- **내적 (Inner Product / Dot Product):** 두 요소(벡터 또는 연속 함수)의 성분들을 곱해 모두 더하거나 적분하여 스칼라 값으로 뽑아내는 연산입니다. 타겟 데이터 안에 특정 기저 함수 패턴이 얼마나 포함되어 있는지 크기를 추출하는 투영(Projection) 과정입니다.
    
- **도메인 (Domain):** 변수가 속하여 변화를 나타내는 기준 축(관점)입니다. 배열의 데이터는 시간 도메인, FFT 변환 후에는 주파수 도메인, 라플라스 변환 후에는 $s$-도메인에 존재합니다.
    
- **이산 (Discrete) vs 연속 (Continuous):** 연속은 무한히 미세하게 쪼갤 수 있는 실수 값의 흐름(미적분 대상)이고, 이산은 일정한 샘플링 간격으로 끊어져 배열의 인덱스처럼 정수로 셀 수 있는 상태(루프 알고리즘 대상)입니다.
    
- **제자리 연산 (In-place Algorithm):** 추가적인 메모리 배열 할당 없이, 원본 배열 내부의 메모리 주소 내에서 원소들끼리 위치를 덮어쓰거나 맞바꾸어 결과를 도출하는 알고리즘 최적화 기법입니다.
    
- **전달 함수 (Transfer Function):** 초기 상태가 $0$일 때, 입력 신호의 라플라스 변환에 대한 출력 신호의 라플라스 변환 비율입니다. 코드로 명령을 내렸을 때 시스템이 어떻게 반응할지를 결정하는 시스템 고유의 수학적 뼈대입니다.
    
- **극점 (Pole)과 영점 (Zero):** 전달 함수의 분모를 $0$으로 만드는 복소수 값이 극점이며, 시스템의 발산(폭주) 여부 즉, 안정성을 결정합니다. 분자를 $0$으로 만드는 값은 영점으로 시스템 반응 파형에 영향을 줍니다.
    
- **특성 방정식 (Characteristic Equation):** 피드백 제어가 적용된 폐루프 시스템에서 전체 전달 함수의 분모를 $0$으로 두는 방정식($1 + C(s)H(s) = 0$)입니다. PID 제어 튜닝을 통해 이 방정식의 해(새로운 극점)를 음수 영역으로 이동시킵니다.
    
- **폐루프 (Closed-loop) / 피드백 (Feedback):** 제어기가 일방적으로 명령만 내리는 것이 아니라, 센서를 통해 현재 시스템 상태를 읽어와 목표값과의 오차를 구하고, 이를 다음 제어 명령에 반영하는 순환 구조입니다.
